1.
```
Docker adalah semacam container yang digunakan untuk membungkus aplikasi ( termasuk package, setting, maupun dependencies ) dan menjalankannya tanpa berada di perangkat sebenarnya. Di Docker juga terdapat image yang dapat kita akses, sama seperti kita menggunakan aplikasi biasa di windows ( OS perangkat ). 
```
2. 
```
Virtual machine termasuk berat karena membutuhkan resource dari perangkat kita, jika kita menggunakan seperti Oracle VM Virtualbox, disitu bisa disetting seberapa kuat komputer virtual yang akan kita buat. Dari sini artinya sama saja dengan kita menjalankan komputer dalam komputer yang akan menjadi kerja cpu maupun ram dan lainnya menurun. Sedangkan untuk docker, ini hanya menjalankan operating system atau OS, sehingga berjalan lebih ringan, meskipun pada prakteknya sama - sama berat juga terlebih pada laptop saya yang spesifikasinya kurang mumpuni. 
```

```
5. 
```
Bisa digunakan docker compose, dimana menggunakan yaml file untuk mengatur layanannya seperti create dan start layanan langsung dari sini menggunakan command line tentunya.
